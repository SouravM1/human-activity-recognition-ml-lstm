# 🏃 Human Activity Recognition using Machine Learning and LSTM

## 📌 Overview

This project implements a **Human Activity Recognition (HAR)** system using smartphone sensor data from the **UCI Human Activity Recognition with Smartphones** dataset. The goal is to automatically classify human activities by comparing the performance of multiple **Machine Learning** algorithms and a **Deep Learning (LSTM)** model.

## 🎯 Objective

To build an accurate activity recognition system that predicts daily human activities from smartphone sensor features and compares traditional Machine Learning techniques with Deep Learning approaches.

## 🤖 Models Implemented

### Machine Learning

* Logistic Regression
* K-Nearest Neighbors (KNN)
* Support Vector Machine (SVM)
* Decision Tree
* Random Forest
* XGBoost

### Deep Learning

* Long Short-Term Memory (LSTM)

## 📊 Activities Classified

* 🚶 Walking
* ⬆️ Walking Upstairs
* ⬇️ Walking Downstairs
* 🪑 Sitting
* 🧍 Standing
* 🛏️ Laying

## 📂 Dataset

The project uses the **Human Activity Recognition with Smartphones** dataset from Kaggle:

https://www.kaggle.com/datasets/uciml/human-activity-recognition-with-smartphones

The dataset contains **561 engineered features** extracted from smartphone accelerometer and gyroscope signals and is divided into predefined training and testing sets.
